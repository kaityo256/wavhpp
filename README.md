# Single C++ Header Library for WAV format

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

## Summary

Sample source codes to make a PCM file (440kHz, two sec).

## Usage

    make
    ./a.out

It will make a file "test.wav".
